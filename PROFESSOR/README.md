# PROFESSOR Android App

تطبيق أندرويد يحوّل واجهة `professor.html` إلى زر عائم مثل ماسنجر.

## الميزات

- اسم التطبيق: `PROFESSOR`.
- الشعار من الصورة المرفقة.
- عند فتح التطبيق، وبعد منح صلاحية الظهور فوق التطبيقات، يظهر زر عائم دائري.
- الضغط على الزر العائم يفتح واجهة كبيرة تحتوي صفحة التداول.
- زر `تصغير` يعيد الواجهة إلى الزر العائم.
- اختصار Quick Settings يمكن إضافته بجانب أزرار الواي فاي والبلوتوث.

## البناء

افتح مجلد `PROFESSOR` في Android Studio، ثم اختر:

```text
Build > Build APK(s)
```

إذا كان لديك Gradle أو Gradle Wrapper متوفرًا لاحقًا، يمكنك البناء من سطر الأوامر:

```powershell
gradle assembleDebug
```

سيظهر ملف APK عادة هنا:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## التشغيل على الهاتف

1. ثبّت التطبيق.
2. افتحه أول مرة.
3. امنح صلاحية `الظهور فوق التطبيقات`.
4. افتحه مرة أخرى، وسيظهر الزر العائم.
5. لإضافة الاختصار بجانب الواي فاي والبلوتوث:
   - في أندرويد 13 وما بعده استخدم زر الإضافة داخل شاشة الإعداد الأولى.
   - أو افتح لوحة الاختصارات، اضغط تحرير، ثم أضف `PROFESSOR` يدويًا.

ملاحظة: أندرويد لا يسمح للتطبيقات بوضع اختصار Quick Settings تلقائيًا بدون موافقة المستخدم.
